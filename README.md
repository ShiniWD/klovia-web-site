# Klovia Web Solutions - Landing Page Profesional

Una landing page moderna y profesional para Klovia Web Solutions, agencia de desarrollo web especializada en soluciones "llave en mano".

## 🎨 Características del Diseño

- **Paleta de Colores**: Basada en el gradiente del logo Klovia (cian a violeta)
- **Diseño Minimalista**: Estético "aireado" con sombras suaves y esquinas redondeadas
- **Totalmente Responsiva**: Adaptada para escritorio, tablet y móvil
- **Accesible**: Con etiquetas ARIA y buen contraste de texto
- **Optimizada**: Código limpio y carga rápida sin librerías pesadas

## 🛠️ Tecnologías Utilizadas

- **HTML5 Semántico**: Estructura accesible y bien organizada
- **CSS3 Moderno**: Variables CSS, Grid, Flexbox y animaciones suaves
- **JavaScript Vanilla**: Interacciones ligeras sin dependencias externas
- **Tipografía Inter**: Fuente moderna y profesional de Google Fonts

## 📁 Estructura del Proyecto

```
klovia-web-solutions/
├── index.html          # Estructura HTML semántica
├── styles.css          # Estilos CSS con variables del logo
├── script.js           # Interacciones JavaScript
└── README.md           # Documentación del proyecto
```

## 🚀 Secciones de la Página

1. **Navegación Fija**: Logo y menú responsive con enlaces suaves
2. **Hero Section**: Título impactante con mockup de laptop
3. **Servicios**: Tres tarjetas con iconos y descripciones
4. **Cómo Funciona**: Proceso en 3 pasos visuales
5. **Diferenciación**: Sección con gradiente y texto blanco
6. **CTA Final**: Formulario de contacto funcional
7. **Footer**: Logo pequeño y redes sociales

## ✨ Interacciones JavaScript

- Menú móvil animado con hamburguesa
- Scroll suave entre secciones
- Animaciones al hacer scroll (Intersection Observer)
- Efecto typing en el título hero
- Formulario con validación y notificaciones
- Efectos hover en tarjetas
- Parallax sutil en imagen hero

## 📱 Responsividad

- **Desktop**: Layout en grid con 2 columnas en hero
- **Tablet**: Adaptación gradual de espaciados
- **Móvil**: Menú hamburguesa y layout de una columna

## 🎯 Puntos Clave del Diseño

- **Colores Brand**: Cian (#00D4FF) y Violeta (#8B5CF6) del logo
- **Gradientes**: Uso consistente del gradiente principal
- **Sombras**: 3 niveles (soft, medium, large) para profundidad
- **Bordes Redondeados**: 12px y 16px para consistencia visual
- **Transiciones**: Cubic-bezier suaves para todas las interacciones

## 🔧 Personalización

Los colores están definidos como variables CSS en `:root`:

```css
--primary-cyan: #00D4FF;
--secondary-violet: #8B5CF6;
--gradient-primary: linear-gradient(135deg, #00D4FF 0%, #8B5CF6 100%);
```

## 📈 Optimizaciones

- **Performance**: Debouncing en eventos scroll
- **Accesibilidad**: ARIA labels y focus states
- **SEO**: Meta tags descriptivos y semántica HTML5
- **Print**: Estilos optimizados para impresión
